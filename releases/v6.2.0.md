# v6.2.0 — GitHub Commander Skills, Shipped Skills Auto-Discovery & Wiki Drift Detector

## ✨ Highlights

- **GitHub Commander skill system** — 8 shipped workflow files teaching agents structured issue triage, PR review, milestone sprints, and security/quality/performance audits
- **Shipped skills auto-discovery** — `memory://skills` now scans the package's own `skills/` directory alongside user-configured paths
- **Wiki Drift Detector** — New GitHub Action that audits wiki documentation for accuracy on every code PR
- **~140 new E2E tests** — 8 new Playwright spec files + comprehensive Code Mode API surface validation
- **Bug fixes** — `restore_backup` structured errors, `team_link_entries` self-loop guard

## Added

- 7 new Playwright E2E specs: `zod-sweep`, `boundary`, `errors-extended`, `numeric-coercion`, `help-resources`, `integration-workflows`, `streamable-http` (~115 tests)
- `codemode-groups.spec.ts` E2E spec: per-group Code Mode API surface validation (25 tests)
- E2E test helpers: `callToolRaw`, `getBaseURL`, `expectHandlerError`
- `TEAM_DB_PATH` in Playwright test server config for full team tool E2E coverage
- **GitHub Commander** skills: 8 workflow files with validation gates, security scanning, journal trails, and HITL checkpoints
- **Shipped skills auto-discovery**: `memory://skills` scans bundled `skills/` directory
- **Wiki Drift Detector**: `.github/workflows/wiki-drift-detector.md` flags outdated wiki docs on PRs

## Fixed

- `restore_backup` now returns structured handler error (not MCP-level `isError: true`) when `confirm` is omitted
- `team_link_entries` now blocks self-referential links, matching personal `link_entries` behavior

## Changed

- `package.json` `files` field: npm tarball ships `dist/`, `skills/`, `LICENSE`, `README.md`
- `onnxruntime-web` overridden with empty stub — eliminates 90 MB unused browser WASM runtime
- `sharp` overridden with empty stub — removes unused image processing dependency

## Dependencies

- `trufflesecurity/trufflehog` 3.93.8 → 3.94.0
- `docker/scout-action` 1.20.2 → 1.20.3
- `github/codeql-action` 4.33.0 → 4.34.1
