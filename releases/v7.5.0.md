# Release v7.5.0

## Highlights

- **Hush Protocol Integration**: A machine-actionable team communication system featuring `team_pass_flag` and `team_resolve_flag` tools.
- **Agent Skill Expansion**: Added `github-copilot-cli` skill for interactive, terminal-native code reviews.
- **Pre-Push Automation**: Introduced `copilot-audit` workflow enabling pre-push review loops.
- **Analytics & Consistency**: Fixed key statistical grouping bugs and resolved E2E transport timeouts in SDK detach stages.

## Added

- Integrated `github-copilot-cli` skill for interactive, terminal-native code reviews.
- Added `copilot-audit` pre-push review loop for evaluating uncommitted changes via Copilot.
- Introduced a machine-actionable team communication system (Hush Protocol).
- Integrated active flags summary and added `localTime` for chronological grounding to Briefing.

## Changed

- Renamed the ambiguous `Matrix Density` insight label to `Relationship density`.
- Routine Dependency Updates (better-sqlite3, globals, simple-git, typescript-eslint, diff, trufflehog, etc.).

## Fixed

- Added missing `importanceScore` property to `JournalEntry` interface.
- Fixed early `.slice()` truncation dropping high-relevance results.
- Fixed SQLite `%Y-Q` quarterly grouping calculation by explicitly deriving months.
- Strictly enforcing `MAX_QUERY_LIMIT` (500) across Team Searches.
- Hardened `teamCollaborationResource` return payload structure.
- Fixed SQLite mapping bug casting `autoContext` JSON strings to booleans.
- Applied date range filter correctly for causal relationship metrics in `get_statistics`.
- Resolved an E2E testing timeout in `session-advanced.spec.ts` by detaching SDK transport correctly.

### Installation

Update your MCP settings to point to the new OCI identifier or run via npx:

```json
{
  "mcpServers": {
    "memory-journal": {
      "command": "npx",
      "args": ["-y", "memory-journal-mcp@7.5.0"]
    }
  }
}
```
