# v7.5.0 - Hush Protocol & Copilot Workflows

## Highlights

- **Hush Protocol Integration**: Added a robust, machine-actionable team communication system using memory flags (`pass_team_flag`, `resolve_team_flag`, `memory://flags`), enabling synchronous and asynchronous team coordination directly within the core journal context.
- **Enhanced Agent Capabilities**: Integrated new `github-copilot-cli` skills and `copilot-audit` workflows, enabling terminal-native code reviews and intelligent pre-push audits.
- **Briefing Maturation**: Added local-time awareness and active-flag summaries to session briefings originally guiding agents' immediate focus.
- **Database & Analytic Stability**: Overhauled nested payload structure handling, schema truncations, limits enforcements across Team Searches, and SQLite quarter projections.

### Added

- **Skills**: Integrated `github-copilot-cli` skill for interactive, terminal-native code reviews.
- **Workflows**: Added `copilot-audit` pre-push review loop for evaluating uncommitted changes via Copilot.
- **Hush Protocol**: Introduced a machine-actionable team communication system (resources, tools, & models).
- **Briefing**: Integrated active flags summary and added `localTime` for chronological grounding.

### Changed

- **Dependency Updates**: Bumped `better-sqlite3` to v12.9.0, `globals` to v17.5.0, and `simple-git` to v3.36.0.
- **Briefing**: Renamed the ambiguous `Matrix Density` insight label to `Relationship density`.

### Fixed

- **Types**: Added missing `importanceScore` property to `JournalEntry` interface.
- **Search**: Fixed early `.slice()` truncation that dropped high-relevance results prior to importance sorting.
- **Analytics**: Fixed SQLite `%Y-Q` quarterly grouping calculation by explicitly deriving months.
- **Constraints**: Fixed raw limit evaluation by strictly enforcing `MAX_QUERY_LIMIT` (500) across Team Searches.
- **Consistency**: Hardened `teamCollaborationResource` return payload structure.
- **Hush Protocol**: Fixed SQLite mapping bug inadvertently casting `autoContext` JSON strings to booleans.
